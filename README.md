## Built With

- React & Redux
- Rails for API
- Enzyme & Jest for Testing
- D3.js for Charting

## Deployment

1. Clone the repo
2. run `npm run i` and `npm start`
3. Open `http://localhost:3000/`
4. Enjoy :)

## Testing

1. Run `npm i`
2. Run `npm run test a`
3. `10 passed, 10 total` will be shown. :)
